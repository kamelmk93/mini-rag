# mini-rag

## Install Python 3.8 (if not already installed)
```bash
sudo apt update
sudo apt install python3.8 python3.8-venv python3.8-distutils
```

### Verify Python 3.8 Installation
```bash
python3.8 --version
```

## Virtual Environment
### create
```bash
python3.8 -m venv mini-rag-app
cd mini-rag-app
```
### Activate
```bash
source bin/activate
```
